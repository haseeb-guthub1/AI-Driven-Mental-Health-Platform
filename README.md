# 🧠 AI-Driven Mental Health Platform

An AI-powered mental health platform built with **Django REST Framework** and **React** that helps users monitor their emotional well-being, connect with professional coaches, and receive personalized AI-based guidance.

---

## 📖 Overview

The AI-Driven Mental Health Platform is designed to improve mental wellness by combining artificial intelligence with professional coaching. Users can track their emotions, manage therapy sessions, receive AI-powered recommendations, and access mental health resources through a secure and user-friendly interface.

---

## ✨ Features

- 🔐 Secure User Authentication (JWT)
- 🤖 AI-Powered Mental Health Guidance
- 😊 Emotion & Mood Tracking
- 👨‍⚕️ Human Coach Management
- 💬 Chat & Session Logging
- 📅 Appointment & Session Scheduling
- 📊 User Dashboard
- 📚 Mental Health Resources
- 🔔 Notification System
- 📁 Resource Upload & Management
- 📈 Progress Monitoring
- 🛡️ Role-Based Access Control

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/AI-Driven-Mental-Health-Platform.git
```

Navigate to the project folder:

```bash
cd AI-Driven-Mental-Health-Platform
```

---

## ⚙️ Backend Setup

Navigate to the backend:

```bash
cd backend/FYP_Backend-main
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate it:

### Windows

```bash
.venv\Scripts\activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

Apply migrations:

```bash
python manage.py migrate
```

Run the server:

```bash
python manage.py runserver
```

Backend runs at:

```
http://127.0.0.1:8000/
```

---

## 💻 Frontend Setup

Open another terminal.

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Frontend runs at:

```
http://localhost:5173/
```
